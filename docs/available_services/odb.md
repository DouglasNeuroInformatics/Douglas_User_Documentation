# Open Data Bank
The Open Data Bank (https://databank.douglasneuroinformatics.ca) is an open-source, web-based platform specifically designed for managing, versioning, and sharing tabular datasets.  The core features include;

## Selective Variable Sharing:
Exercise granular control over who can access specific variables within your datasets, ensuring that sensitive columns are protected.
## Tabular Data Management:
Seamlessly upload datasets in standard CSV or Excel formats.
## Automated Versioning:
Maintain a perfect record of data changes with automatically generated changelogs on every update.
## Centralized Collaboration:
Store all lab datasets in a single, secure location to facilitate collaboration between investigators.
